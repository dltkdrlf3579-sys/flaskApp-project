SSO가 아직 설정되지 않았습니다

운영용 IdP 주소/리다이렉트 URL이 예시(example) 값으로 남아있어 인증을 진행할 수 없습니다.

개발 환경에서 확인하고 싶다면
  1. config.ini → [SSO] sso_enabled = false
  2. config.ini → [SSO] dev_simulate_flow = true
  3. 페이지 재접속 → 샘플 로그인 화면(/sso/dev-login)에서 Username 입력
운영 환경에서 연동하려면
  1. [SSO] sso_enabled = true, cert_enabled = true
  2. idp_entity_id, idp_client_id, sp_redirect_url 등을 실제 값으로 설정
  3. cert/idp.cer 인증서 파일 배치
  4. IdP에 리다이렉트 URI(…/sso/callback) 등록
